Tasks are clearer

  • Struct futures::task::Context
    • Represents a task's context
    • Context::executor() -> &mut Executor provides access to the task's executor
    • Context::spawn(...) schedules a new operation on the executor
    • Context::waker() -> &Waker provides access to the mechanism to awake the task
  • Struct futures::task::Waker
    • Waker::wake()
    • Replaces Task::notify()
