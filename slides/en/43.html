Input and output operations

  • Some helper functions allow composing operations, returning Futures
    • copy: copies all of the data from a reader into a writer
    • flush: sends all of the queued data
    • read_exact: reads a certain number of bytes
    • read_to_end: reads all of the bytes from a reader
    • write_all: writes all of the bytes in a provided buffer
    • shutdown: closes a writer
  • A function that retorns a Stream
    • lines: reads line by line, producing Strings
